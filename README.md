# fcmMessage
Simple app showing connect with firebase DB + messaging

此專案為教學文章用，文章來源：
https://ioswithnick.blogspot.com/2020/05/firebase.html

<b>安裝 Install -</b>
專案不包含GoogleService-Info.plist 和 Pods，記得從Firebase下載plist和重新執行pod install
